== Description ==

This will allow administrators to prune out old users that have never contributed to the blog with either a comment or a post.

== Installation ==

1. Upload 'prune-users.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the plugin in the Settings menu
